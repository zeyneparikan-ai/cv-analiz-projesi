# 📄 CV Analiz Projesi

NLP teknolojisi kullanan otomatik CV analiz sistemi.

## 🚀 Özellikler
- CV'den otomatik bilgi çıkarma
- NLP ile metin analizi
- FastAPI ile web arayüzü

## 🛠️ Kullanılan Teknolojiler
- Python
- spaCy (NLP)
- FastAPI

## ▶️ Çalıştırmak İçin
pip install -r requirements.txt
python app.py
